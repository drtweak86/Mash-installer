 
# Goblin-Tiered Software List

## S-Tier (Best)
| Category       | Software               | Reasoning                                  |
|----------------|-------------------------|--------------------------------------------|
| Terminal       | Kitty                   | GPU-accelerated, fast, ligatures.          |
| Shell          | Zsh + Starship          | Customizable, fast, Nerd Font support.     |
| File Manager   | eza                     | Icons, Git integration, tree view.         |
| Text Editor    | Helix                   | Modern Vim-like, written in Rust.          |
| Git Client     | Lazygit                 | TUI Git client (no mouse needed).          |
| Process Viewer | btop                    | Prettier, more info than htop.             |
| Browser        | Brave                   | Sync, crypto, ad-block.                    |
| Media Player   | MPV                     | Lightweight, scriptable.                   |
| HTPC           | Kodi                    | Untouchable.                               |
| VPN            | WireGuard               | Faster, simpler than OpenVPN.              |
| Firewall       | nftables               | Modern, faster than iptables.              |
| Backup         | rclone + borg           | Cloud + local snapshots.                   |

| Category       | Software               | Reasoning                                  |
|----------------|-------------------------|--------------------------------------------|
| Terminal       | Alacritty               | Good, but Kitty is better.                 |

| Git Client     | tig                     | Good, but Lazygit is easier.               |
| Process Viewer | glances                 | Web UI, but btop is prettier.              |
